Special Instructions for Running and Checking the Imported `.fig` File

1. Introduction

This document provides the necessary instructions to run and evaluate the design contained in the submitted `.fig` file using Figma. It outlines the steps required to properly access, review, and validate the design and its functionality.

---

2. Software Requirement

* Figma (Web or Desktop Application)

---

3. Importing the `.fig` File

* Open the Figma application.
* Navigate to the dashboard.
* Select the **Import** option.
* Upload the provided `.fig` file.
* Open the file to begin reviewing the design.

---

4. Running the Design (Prototype Mode)

* Click on the **Present (Play)** button located at the top-right corner.
* The prototype preview will open in a new view.
* Interact with the design to test navigation and user flow.
* All Pages are connected in relevant areas.

---

5. Reviewing the Design Structure

* Use the left-side panel to examine frames, layers, and components.

---
